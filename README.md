# Task 18: DOM Handling

## Overview
This task demonstrates basic DOM manipulation in JavaScript.

The page includes:
- A greeting heading that updates with the entered name.
- Four colored boxes that change their background color when clicked.

## Files
- `index.html` — page structure and UI elements.
- `style.css` — styling for the page and boxes.
- `script.js` — JavaScript logic for updating the greeting and coloring boxes.

## How it works
1. Enter a name in the input field.
2. Click the `Greet` button.
3. The heading text updates to `Hello, <name>`.
4. Click any of the color boxes to set that box's background to its label color.

## Notes
- Each box uses its displayed text as the color value.
- The greeting input uses a click event listener to update the heading.

## Usage
Open `index.html` in a browser to try the interaction.
